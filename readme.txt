== Description ==

Add a "Go to top" link to your posts, this is a wordpress plugin which base on	<a href="http://jquery.com/" target="_blank">jQuery</a>, <a href="http://gsgd.co.uk/sandbox/jquery/easing/" target="_blank">jQuery Easing</a> and <a href="http://blog.ph-creative.com/post/jquery-plugin-scroll-to-top-v3.aspx" target="_blank">Scroll to Top v3</a>

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
